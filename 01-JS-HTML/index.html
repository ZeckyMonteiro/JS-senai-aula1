Atalhos do Vs Code

Atalhos gerais

Vamos ver alguns atalhos legais

ctrl+espaço: chama o autocompletar de um contexto. (procura a imagem na pasta,pra colocar no código)

alt+z: quebra automática de linha

alt+setas cima/baixo: move a seleção para baixo no arquivo. se nada estiver selecionado,movimenta a linha que o cursor está.

shift+alt+setas cima/baixo: copia a seleção para cima/baixo no arquivo. se nada estiver selecionado, copia a linha em que o cursor está.
